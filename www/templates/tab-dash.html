Welcome to DerpChat

DerpChat is an anonymous chat application that allows you to express yourself to your homies and to future homies.

DerpChat is built on Ionic and AngularJS and communicates with an AWS webserver using node and the Socket.io library. The ionic application can be found here: Github Repo

To edit the content of each tab, edit the corresponding template file in www/templates/. This template is www/templates/tab-dash.html
